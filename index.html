Project 01

Flask 웹 게시판 애플리케이션

사용자 인증, 게시글 작성, 파일 업로드를 하나의 흐름으로 묶은 웹 게시판입니다. 로그인 이후 게시글과 첨부 파일을 관리할 수 있도록 기본 CRUD 구조를 구현했습니다.

핵심 기능

  • 회원가입과 로그인 기반 사용자 세션 관리
  • 게시글 작성, 수정, 삭제를 포함한 CRUD 흐름
  • 이미지와 영상 업로드를 고려한 게시글 작성 화면

구현 포인트

Flask-Login과 SQLAlchemy를 활용해 사용자 인증과 데이터 저장 책임을 분리했습니다. 화면 단에서는 게시판, 로그인, 회원가입, 작성 페이지가 자연스럽게 이어지도록 구성했습니다.

Project 03

포트홀 감지 및 관리 시스템

YOLO 모델과 OpenCV를 활용해 영상에서 포트홀을 감지하고, 감지 이미지를 데이터베이스에 기록하는 관리 시스템입니다. 인증된 사용자가 결과를 조회하고 관리할 수 있도록 구성했습니다.

핵심 기능

  • 회원가입과 로그인으로 보호되는 대시보드
  • YOLO 기반 포트홀 감지와 감지 이미지 저장
  • 감지 시간과 이미지 경로를 데이터베이스에 기록

구현 포인트

실시간 영상 처리 루프에서 감지 결과를 이미지로 저장하고, 사용자 ID와 함께 DB에 연결했습니다. 인증, 감지, 조회 흐름이 하나의 서비스처럼 동작하도록 설계했습니다.
